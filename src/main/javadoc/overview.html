TrueZIP is a Java based plug-in framework for virtual file systems (VFS) which provides transparent access to archive files as if they were just plain directories.

As an API, TrueZIP provides transparent, uniform, thread-safe, read/write access to archive files as if they were just plain directories in a file system path.

As a framework, TrueZIP provides the interfaces and classes to write file system drivers which plug-in to its federated file system space.
