Class TFileWriter

  • java.lang.Object

    @Immutable
@CleanupObligation
public final class TFileWriter
extends OutputStreamWriter
    A replacement for the class FileWriter for writing plain old files or entries in an archive file. Mind that applications cannot write archive files directly - just their entries!
